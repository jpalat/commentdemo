CBRE Customer Service Analysis Tool
---

Description
--
CBRE surveys internal customers for open ended feedback on service offerings.  This project provides a sentence transfomer that will classify incoming comments as Technical, Performance, User Experience (UX), and Data/Security.  

Sentence Transformer for huggingface is finetuned to based on the labeled data provided in labeled_data.csv.

A simple web application is provided to allow users to add new comments and see how they are classified.
